йаПосылалка v1.3

Скрипт анонимной отправки почты на php через функцию send (требуется sendmail или аналог).

  • Всего 1ин файл
  • Ajax интерфейс
  • Многопоточная отправка с историей(Есть подгрузка списка из файла)
  • Макросы в тексте письма и полях (есть безлимитные дополнительные поля макросов)
  • Сохранение и загрузка шаблона письма
  • Эксперементальная функция переноса картинок из ссылок в аттач
  • Эксперементальная функция обхода фильтров через реверсинг слов с нормализацией средствами CSS
